#### Manual proxy
A proxy string can be provided in the fingerprint as either: 
- `{host}:{port}:{username}:{password}`
- `http://{username}:{password}@{host}:{port}`


#### Proxy Generation
Alternatively a proxy can be generated from information provided in the fingerprint to get a proxy from our high quality pool.

 - **Proxy Type** (datacenter/residential)
 - **Country ID** (lowercase [alpha-2 country code](https://www.iban.com/country-codes))
