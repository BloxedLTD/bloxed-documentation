**Proxies**
A proxy string can be provided in the format {host}:{port}:{username}:{password}

Alternatively a **proxy** generation data can be provided in the fingerprint to generate a proxy at scrape time.
This contains the following information:
 - **Proxy Type** (isp/datacenter/residential)
 - **Country ID** (gb/us)
 - **City Alias** (lon/man)

A **user agent** string can also be provided.
