title: 'Data Cleaning'

---

For all variants of the data cleaning module:
- **Input** is a list of scopes to the data you would like to transform.
- **Output** is a list of output key that you would like to write the new data to. (**one-to-one** relationship with inputs).
- **Recurse** allows for recursion through every part of the element (from the scope provided).
- **Error On Fail** lets you choose if failure of a data clean (i.e an invalid index) will cause an error to occur or just be ignored.

<Info>
If no output is given, the new data will overwrite the original input key
</Info>

## **Types**
### **ToNumber**
Converts a string to an int.
```json
    {
        "module": "data_cleaning",
        "variant": "to_number",
        "input": [
            "title"
        ],
        "output": []
    }
```
### **ToString**
Converts an int to a string.
```json
    {
        "module": "data_cleaning",
        "variant": "to_string",
        "input": [
            "title"
        ],
        "output": []
    }
```
## **Retrieve**
### **Get**
Fetches data from the scope provided.
```json
    {
        "module": "data_cleaning",
        "variant": "get",
        "input": [
            "title"
        ],
        "output": [],
        "recurse": true,
        "error_on_failure": true
    }
```
Fetches the length of the item at the scope provided
### **Length**
Fetches the length of the item at the scope provided.
```json
    {
        "module": "data_cleaning",
        "variant": "length",
        "input": [
            "title"
        ],
        "output": []
    }
```
## **Transformation**
### **Flatten**
'Squashes' lists at the scope provided into a single one.
```json
    {
        "module": "data_cleaning",
        "variant": "flatten",
        "input": [
            "title"
        ],
        "output": []
    }
```
### **Split By**
Splits the list at the scope on the **Character** provided.
```json
    {
        "module": "data_cleaning",
        "variant": "split_by",
        "character": "q",
        "input": [
            "title"
        ],
        "output": []
    }
```
### **Replace**
**Original** determines the character to be replaced with **new**.
```json
    {
        "module": "data_cleaning",
        "variant": "replace",
        "original": "q",
        "new": "a",
        "input": [
            "title"
        ],
        "output": []
    }
```
