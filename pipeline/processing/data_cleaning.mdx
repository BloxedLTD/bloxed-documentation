---
title: 'Data Cleaning'
description: 'Operate on the extracted data to tidy it up'
icon: 'shower'
---

For all variants of the data cleaning module:
- **Input** is a list of scopes to the data you would like to transform.
- **Output** is a list of output key that you would like to write the new data to. (**one-to-one** relationship with inputs).
- **Recurse** lets your recurse through every part of the elements (e.g. nested arrays).
- **Error On Fail** lets you choose if failure of a data clean will cause an error to occur or just be ignored.

<Info>
If no output is given, the new data will overwrite the original input key
</Info>

#### **Get**
```json
    {
        "module": "data_cleaning",
        "variant": "get",
        "input": [
            "title.[0]"
        ],
        "output": []
    }
```
#### **Split At**
**Index** determines the value to split at.
```json
    {
        "module": "data_cleaning",
        "variant": "split_at",
        "index": 1,
        "input": [
            "title"
        ],
        "output": []
    }
```
#### **Slice**
**Start**/**End** determines the start and end of the slice.
```json
    {
        "module": "data_cleaning",
        "variant": "slice",
        "start": 1,
        "end": 5,
        "input": [
            "title"
        ],
        "output": []
    }
```

**Character** determines the character to be removed.
```json
    {
        "module": "data_cleaning",
        "variant": "strip",
        "character": "a",
        "input": [
            "title"
        ],
        "output": []
    }
```
#### **Split By**
**Character** determines the character to split on.
```json
    {
        "module": "data_cleaning",
        "variant": "split_by",
        "character": "q",
        "input": [
            "title"
        ],
        "output": []
    }
```
#### **Replace**
**Original** determines the character to be replaced with **new**.
```json
    {
        "module": "data_cleaning",
        "variant": "replace",
        "original": "q",
        "new": "a",
        "input": [
            "title"
        ],
        "output": []
    }
```
