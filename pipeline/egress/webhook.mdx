### Output:
Below are example outputs from the scrape. 

All fields will either contain extracted data or an error object. (If in/out the whole task is sent back to the api and just the relevent data is returned. For other egress methods, just the relevent data is returned.)

**Output** contains the mappings described in the extraction rules during task creation.
These map to a list that contains the extracted data.

#### Error Object:
```go
    type OutputError struct {
        StatusCode int    `json:"status_code"`
        Message    string `json:"message"`
    }
```

**Error** will contain any error that has occurred from egress modules. Otherwise no data should show.
