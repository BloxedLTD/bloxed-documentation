## Setup your task

Learn how to create a task pipeline to scrape a site
```json
{
    "pipeline": {
        "scraping": {
            "url": "https://www.amazon.co.uk/tv/s?k=tv",
            "fingerprint": {
                "proxy_type": "residential",
                "country_id": "gb"
            },
            "instructions": [
                {
                    "action": "extraction",
                    "selector": "[data-cy='price-recipe'] .a-offscreen|text",
                    "output": "price"
                }
            ]
        },
        "processing": [
            {
                "module": "data_cleaning",
                "variant": "replace",
                "old": "Â£",
                "new": "",
                "input": "price.[]"
            },
            {
                "module": "data_cleaning",
                "variant": "replace",
                "old": ".",
                "new": "",
                "input": "price.[]"
            },
            {
                "module": "data_cleaning",
                "variant": "to_number",
                "input": "price.[].[]"
            }
        ]
    }
}
```
